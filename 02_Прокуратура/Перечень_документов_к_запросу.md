# Перечень документов для прокуратуры

## Кадровые документы

- трудовые договоры;
- дополнительные соглашения;
- приказы о приеме;
- приказы о переводе;
- приказы об увольнении;
- заявления работников;
- личные карточки, если запрошены;
- табели учета рабочего времени за запрошенный период.

## Документы по оплате труда

- положение об оплате труда;
- штатное расписание;
- расчетные листки;
- ведомости начисления;
- платежные ведомости/реестры;
- банковские выписки;
- расчет компенсации за задержку.

## Документы по причинам задолженности

- договор с заказчиком;
- акты выполненных работ;
- счета;
- переписка с заказчиком;
- претензии заказчику;
- документы о дебиторской задолженности.

## Документы по простою, если введен

- докладная записка;
- приказ о простое;
- уведомления работникам;
- лист ознакомления;
- табель с кодами простоя;
- сведения, направленные через «Работа России».
