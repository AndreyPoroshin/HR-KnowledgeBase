# Чек-лист перед введением простоя

- [ ] Есть объективная причина простоя.
- [ ] Причина подтверждается документами.
- [ ] Определены работники/подразделения.
- [ ] Определена дата начала простоя.
- [ ] Подготовлена докладная записка.
- [ ] Подготовлен приказ.
- [ ] Работники уведомлены.
- [ ] Есть лист ознакомления или подтверждение электронного ознакомления.
- [ ] Определен порядок нахождения работников во время простоя.
- [ ] Настроено табелирование.
- [ ] Подготовлена подача сведений через «Работа России».
- [ ] Подготовлены документы для прокуратуры/ГИТ, если есть проверка.
