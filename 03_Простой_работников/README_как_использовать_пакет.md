# Пакет документов для введения простоя работников

Состав:
1. Служебная записка / обоснование простоя.
2. Приказ о введении простоя.
3. Реестр работников.
4. Уведомление работнику.
5. Лист ознакомления / акт отказа.
6. Памятка по оплате и табелю.
7. Черновик сведений для портала «Работа России».
8. Приказ об окончании простоя.
9. Пояснение для прокуратуры/ГИТ.
10. Чек-лист кадровика.

Ключевая позиция: при отсутствии заказов, финансирования или возможности обеспечить работой безопаснее оформлять простой по вине работодателя с оплатой не ниже 2/3 среднего заработка.

Перед применением заполнить реквизиты организации, даты, Ф.И.О., должности, подразделения и приложить доказательства причин простоя.
