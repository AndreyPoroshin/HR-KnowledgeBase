# Уведомление о простое через «Работа России»

## Когда нужно

Если у работодателя возникает простой, сведения направляются через портал «Работа России» в порядке, установленном для информирования органов службы занятости.

## Что подготовить

- приказ о простое;
- список работников;
- даты начала и предполагаемого окончания;
- причина простоя;
- количество работников;
- сведения об организации.

## Практический порядок

1. Войти на портал через учетную запись работодателя.
2. Выбрать раздел для подачи сведений работодателем.
3. Заполнить сведения о простое.
4. Проверить количество работников и даты.
5. Сохранить подтверждение отправки.
6. Подшить подтверждение к кадровым документам по простою.
