# Текущая ситуация

В организации возникла задолженность по заработной плате.

Причина возникновения задолженности:

* отсутствие поступления денежных средств от заказчика;
* наличие дебиторской задолженности;
* недостаточность денежных средств на расчетных счетах организации.

Работодатель не оспаривает наличие задолженности.

Основная задача работодателя:

* сохранить деятельность организации;
* погасить задолженность перед работниками;
* минимизировать риски привлечения к административной и уголовной ответственности;
* обеспечить корректное взаимодействие с прокуратурой и Государственной инспекцией труда.

# Жалобы работников

В июне 2026 года несколько работников уволились и направили обращения в прокуратуру.

Предмет обращений:

* невыплата заработной платы;
* нарушение сроков расчета при увольнении.

---

# Прокурорская проверка

Получен запрос прокуратуры.

Запрашиваются:

* трудовые договоры;
* дополнительные соглашения;
* табели учета рабочего времени;
* расчетные документы;
* приказы;
* сведения о задолженности;
* иные кадровые документы.

Позиция работодателя:

* задолженность признается;
* отсутствует умысел на невыплату заработной платы;
* причиной является отсутствие финансирования со стороны заказчика;
* принимаются меры по погашению задолженности.

---

# Простой работников

Рассматривается введение простоя.

Возможные основания:

* отсутствие финансирования;
* отсутствие объема работ;
* остановка проекта.

Предпочтительная правовая позиция:

Простой по вине работодателя с оплатой не менее 2/3 среднего заработка в соответствии со ст. 157 ТК РФ.

Для простоя подготовлены:

* приказ;
* лист ознакомления;
* уведомления работников;
* рекомендации по табелированию;
* материалы по уведомлению через «Работа России».

---

# Дистанционные работники

В организации имеются дистанционные работники.

Позиция:

Оформление простоя производится в общем порядке с обязательным ознакомлением работников доступными способами связи.

# Ключевые кадровые темы проекта

## Задолженность по заработной плате

Основные вопросы:

* компенсация по ст. 236 ТК РФ;
* расчет при увольнении;
* взаимодействие с прокуратурой;
* взаимодействие с ГИТ;
* документальное подтверждение отсутствия денежных средств.

## Простой

Основные вопросы:

* основания введения;
* порядок оформления;
* табелирование;
* уведомление государственных органов.

## Архив кадровых документов

Изучены вопросы:

* сроки хранения табелей;
* сроки хранения приказов;
* уничтожение кадровых документов;
* оформление актов об уничтожении.

## Отпуска

Изучены вопросы:

* отпуск без сохранения заработной платы;
* отпуск с последующим увольнением;
* отказ в предоставлении отпуска.

## Совместительство

Изучены вопросы:

* работа по совместительству во время отпуска без сохранения заработной платы;
* ограничения по продолжительности рабочего времени.
