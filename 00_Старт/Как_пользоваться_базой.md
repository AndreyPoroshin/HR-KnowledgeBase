# Как пользоваться базой

## 1. Для быстрых действий

Открывать папку по теме и брать файл `Алгоритм_...md` или `Порядок_...md`.

## 2. Для подготовки документов

Открывать папку `templates` или нужную тему, например `03_Простой_работников`.

## 3. Для прокуратуры / ГИТ

Сначала фиксировать факты:

- даты возникновения задолженности;
- размер долга;
- список работников;
- банковские выписки;
- причины отсутствия денег;
- переписку/договоры с заказчиком;
- действия работодателя после возникновения долга.

## 4. Для обновления базы

После изменений выполнить в PowerShell:

```powershell
git add .
git commit -m "Update HR Knowledge Base"
git push
```
