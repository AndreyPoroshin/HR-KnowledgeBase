# Совместительство во время отпуска по основному месту работы

## Ключевой вывод

Отпуск без сохранения заработной платы по основному месту работы сам по себе не отменяет ограничения и правила работы по совместительству.

## Что анализировать

- внутреннее или внешнее совместительство;
- режим рабочего времени;
- продолжительность рабочего дня;
- есть ли ограничения по категории работника;
- не маскирует ли ситуация фактический перевод на полную занятость.

## Практический риск

Если работник длительно работает полный день по совместительству, при этом по основному месту находится в отпуске без сохранения заработной платы, ситуацию нужно оформлять аккуратно и отдельно анализировать.
