# Отпуск без сохранения заработной платы

## Общий принцип

Отпуск без сохранения заработной платы предоставляется по соглашению между работником и работодателем, кроме случаев, когда работодатель обязан его предоставить.

## Практический вывод

Если работник просит отпуск за свой счет с последующим увольнением, работодатель не обязан автоматически соглашаться, если нет обязательного основания.

## Документы

- заявление работника;
- резолюция работодателя;
- приказ при согласовании;
- табель;
- документы об увольнении, если работник подал заявление по собственному желанию.
